# 가드닝다이어리

<br>
기간 : 2021.07.23 ~ 2021.10.22
<br><br>
가드닝 하는 사람들을 위해 성장과정을 기록하고 작물질병진단을 할 수 있는 서비스입니다.
<br><br>

* 적용기술 
  * Back-End  
    * Spring Framework 
    * JAVA 
  * DevOps 
    * MariaDB
    * AWS EC2
    * RDS
  * AI 모델
    * Teachable Machine
  * Front-End 
    * HTML 
    * CSS 
    * Javascript 
    * BootStrap 
  * 기타 
    * Open API
